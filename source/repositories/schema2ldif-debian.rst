Debian Stretch
^^^^^^^^^^^^^^

Add a file named **schema2ldif-release.list** in /etc/apt/sources.list.d/

.. code-block:: shell

   #latest version of schema2ldif
   deb https://public.fusiondirectory.org/stretch-schema2ldif-release/ stretch main
