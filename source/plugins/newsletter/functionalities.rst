Functionalities
===============

* Create Newsletter for a user

Click on Users icon in FusionDirectory

.. image:: images/newsletter-users.png
   :alt: Picture of Users icon in FusionDirectory

Select a user

.. image:: images/newsletter-user.png
   :alt: Picture of User line in FusionDirectory

Go to Newsletter tab

.. image:: images/newsletter-tab.png
   :alt: Picture of Newsletter tab in FusionDirectory

Click on Add Newsletter settings button

.. image:: images/newsletter-add-settings.png
   :alt: Picture of Add Newsletter settings button in FusionDirectory

Fill-in Personal info 

.. image:: images/newsletter-personal-info.png
   :alt: Picture of Newsletter Personal info in FusionDirectory

Click on Apply button bottom right to save your settings 

.. image:: images/newsletter-apply.png
   :alt: Picture of Newsletter Personal info in FusionDirectory
